std::swap (非メンバ関数)(C++11)

template <class charT, class traits>
void swap(basic_regex<charT, traits>& lhs, basic_regex<charT, traits>& rhs);

概要

2 つの正規表現オブジェクトを交換する。

効果

lhs.swap(rhs)

戻り値

なし
