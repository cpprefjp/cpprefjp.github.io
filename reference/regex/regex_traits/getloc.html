std::regex_traits::getloc(C++11)

locale_type getloc() const;

概要

ロケールを取得する。getlocはget localeの略称。

戻り値

imbue()メンバ関数によってすでにロケールオブジェクトが設定されている場合は、そのコピーを返す。そうでない場合、グローバルロケールオブジェクトのコピーを返す。

例

#include <regex>
#include <cassert>

int main()
{
  std::regex_traits<char> traits;

  traits.imbue(std::locale("ja_JP.UTF-8"));
  std::locale loc = traits.getloc();

  assert(loc.name() == "ja_JP.UTF-8");
}
