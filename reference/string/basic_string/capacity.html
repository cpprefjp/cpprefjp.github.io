capacity

size_type capacity() const noexcept;

概要

メモリを再確保せずに格納できる最大の要素数を取得する。

戻り値

メモリを再確保せずに格納できる最大の要素数

例外

投げない

例

#include <iostream>
#include <string>

int main()
{
  std::string s;
  s.reserve(3);

  // 確保したサイズを確認
  std::size_t cap = s.capacity();
  std::cout << cap << std::endl;
}

出力例

3
