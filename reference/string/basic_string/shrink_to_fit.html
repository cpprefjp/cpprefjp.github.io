shrink_to_fit (C++11)

void shrink_to_fit();

概要

領域をコンテナのサイズまで切り詰める

戻り値

なし

備考

capacity()をsize()に縮小させるリクエストを行う。

実装依存の最適化を許可するために、縮小するという動作は仕様上強制されない。

例

shrink_to_fit();

void basic_string::shrink_to_fit() {
  swap(basic_string(*this));
}
