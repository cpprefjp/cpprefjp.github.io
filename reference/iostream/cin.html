cin

namespace std {
  istream cin;
}

概要

cinは、標準入力に対するマルチバイト入力ストリームオブジェクトである。 すなわち、<cstdio>のstdinオブジェクトに結びつけられたストリームオブジェクトである。

例

#include <iostream>
#include <string>

int main()
{
  std::cout << "名前を入力してください: ";

  std::string s;
  std::cin >> s;

  std::cout << "あなたの名前は「" << s << "」ですね。" << std::endl;
}
