デストラクタ (C++11)

~thread();

概要

threadオブジェクトを破棄する。

効果

デストラクタ呼び出し時点でthreadオブジェクトにスレッドが関連付けられている場合、std::terminate()を呼び出してプログラムを終了する。既にjoin操作またはdetach操作済みの（つまり、threadオブジェクトが既にスレッドと関連付けられていない）場合は何もしない。

例

#include <thread>

int main()
{
  {
    std::thread thd([]{ /*...*/ });
    // thdに対してjoin()もdetach()も行わない。

    // thdのデストラクタ呼び出しでプログラム終了!
  }

  return 0;
}
