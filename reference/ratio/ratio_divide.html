ratio_divide (C++11)

namespace std {
  template <class R1, class R2>
  using ratio_divide = …;
}

概要

ratio_divideは、左辺のratioを右辺のratioで除算したratio型を作るエイリアステンプレートである。

効果

除算結果は、ratio<R1::num * R2::den, R1::den * R2::num>という型になる。

例

#include <ratio>

int main()
{
  using r1 = std::ratio<2, 5>;
  using r2 = std::ratio<5, 3>;

  using result = std::ratio_divide<r1, r2>;

  static_assert(result::num == 6, "result num is 6");
  static_assert(result::den == 25, "result den is 25");
}
