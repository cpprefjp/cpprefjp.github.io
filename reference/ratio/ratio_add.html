ratio_add (C++11)

namespace std {
  template <class R1, class R2>
  using ratio_add = …;
}

概要

ratio_addは、2つのratioを加算したratio型を作るエイリアステンプレートである。

効果

加算結果は、ratio<R1::num * R2::den + R2::num * R1::den, R1::den * R2::den>という型になる。

例

#include <ratio>

int main()
{
  using r1 = std::ratio<1, 5>;
  using r2 = std::ratio<2, 5>;

  using result = std::ratio_add<r1, r2>;

  static_assert(result::num == 3, "result num is 3");
  static_assert(result::den == 5, "result den is 5");
}
