data (C++11)

T* data() noexcept;
const T* data() const noexcept;

概要

配列の先頭へのポインタを取得する。

戻り値

arrayクラス内部の組み込み配列変数名がelemsであった場合、elemsを返す。

例外

投げない

備考

0要素の場合(N == 0)、この関数の呼び出しによる挙動は未規定。
