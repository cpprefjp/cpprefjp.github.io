back (C++11)

reference back();
const_reference back() const;

概要

最後尾要素を参照する。

戻り値

a.back()は末尾の要素への参照を返す。もし、aがconstだった場合には、末尾の要素へのconst参照を返す。a.back() は{ auto tmp = a.end(); --tmp; return *tmp; } と同じ結果になる。

備考

要素数が0の場合(N == 0の場合)、この関数呼び出しの効果は未定義である。
