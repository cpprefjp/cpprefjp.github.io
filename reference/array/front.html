front (C++11)

reference front();
const_reference front() const;

概要

先頭要素を参照する。

戻り値

a.front()は先頭の要素への参照を返す。もし、aがconstだった場合には、先頭の要素へのconst参照を返す。

備考

a.front() と *a.begin() は同じ結果になる。 要素数が0の場合(N == 0の場合)、この関数呼び出しの効果は未定義である。
