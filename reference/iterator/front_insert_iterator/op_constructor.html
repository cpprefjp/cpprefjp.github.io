std::front_insert_iterator::コンストラクタ

explicit front_insert_iterator(Container& x);           // (1) C++03
explicit constexpr front_insert_iterator(Container& x); // (1) C++20

概要

front_insert_iteratorオブジェクトを構築する。

効果

xへのポインタをメンバ変数containerに保持する。

  • C++11まで : xへのポインタは、&xで取得する
  • C++14以降 : xへのポインタは、std::addressof(x)で取得する

参照

  • LWG Issue 2324. Insert iterator constructors should use addressof()
