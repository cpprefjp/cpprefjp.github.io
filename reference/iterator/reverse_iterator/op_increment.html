operator++

reverse_iterator& operator++();
reverse_iterator operator++(int);

概要

イテレータをインクリメントする。 reverse_iteratorなので逆方向に進める。

効果

  • 前置インクリメント operator++()：

--current;
return *this;
- 後置インクリメント operator++(int)：

reverse_iterator tmp = *this;
--current;
return tmp;
