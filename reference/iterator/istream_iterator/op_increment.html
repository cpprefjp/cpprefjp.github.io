operator++

istream_iterator<T, CharT, Traits, Distance>& operator++();
istream_iterator<T, CharT, Traits, Distance> operator++(int);

概要

イテレータを進める。 入力ストリームからoperator>>でデータを読み取る。

効果

前置operator++： *in_stream ≫ value; return *this;

後置operator++： istream_iterator<T, CharT, Traits, Distance> tmp = *this; *in_stream ≫ value; return tmp;

※in_streamは、メンバ変数として保持している入力ストリームオブジェクトへのポインタ
