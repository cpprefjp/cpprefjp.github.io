概要

stack が空か否か、つまり要素数が 0 か否かを返す。 内部のコンテナの empty() メンバ関数を呼ぶ。

戻り値

要素数が 0 の場合は true、それ以外の場合は false 。

計算量

定数時間 O(1)。
