push

void push(const value_type& x);
void push(value_type&& x);  // C++11

概要

新しい要素を stack の末尾に追加し、そのインスタンスをxのコピー、もしくはムーブして初期化する。

パラメータ

x: 新しい要素としてコピー、もしくはムーブする値。

効果

void push(const value_type& x): c.push_back(x) void push(value_type&& x): c.push_back(std::move(x))

戻り値

なし

計算量

Container::push_back()と同じ。
