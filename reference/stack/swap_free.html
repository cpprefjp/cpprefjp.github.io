swap(非メンバ関数) (C++11)

namespace std {
  template <class T, class Container>
  void swap(stack<T, Container>& x, stack<T, Container>& y) noexcept(noexcept(x.swap(y)));
}

概要

2つの stack オブジェクトを入れ替える。

効果

x.swap(y)

戻り値

なし

例外

x.swap(y) が例外を投げない場合、この関数は決して例外を投げない。

例

swap(x, y);
