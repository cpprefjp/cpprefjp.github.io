operator<

namespace std {
  template <class T, class Container>
  bool operator< (const stack<T, Container>& x, const stack<T, Container>& y);
}

概要

stack において、左辺が右辺より小さいかの判定を行う。

戻り値

x.c < y.c

例

#include <iostream>
#include <stack>

int main ()
{
  std::stack<int>  x;
  x.push(2);
  x.push(7);
  x.push(1);

  std::stack<int>  y;
  y.push(3);
  y.push(1);
  y.push(4);

  std::cout << std::boolalpha << (x < y) << std::endl;

}

出力

true
