type-alias
<cstddef>

std::size_t

namespace std {
  using size_t = implementation-defined;
}

size_tは、sizeof演算子によって返される符号なし整数型である。

この型はそのほかに、コンテナの要素数やインデックス値を表すために使用される。

例

#include <cstddef>
#include <cassert>

int main()
{
  std::size_t n = sizeof('a');
  assert(n == 1);
}
