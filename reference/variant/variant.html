std::variant(C++17)

namespace std {
  template <class... Types>
  class variant;
}

概要

variantクラスは、格納されうる候補の型リスト (Types...) に含まれる型のオブジェクトを切り替えながら保持する記憶域型である。継承関係にない複数の型を、単一のオブジェクトに代入・切り替えができる。

このクラスは動的メモリ確保は行わず、スタック領域のみを使用する。
