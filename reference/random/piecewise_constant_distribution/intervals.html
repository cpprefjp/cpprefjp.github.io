intervals (C++11)

vector<result_type> intervals() const;

概要

区間の数列を取得する。

戻り値

保持している区間列を返す。

例

#include <iostream>
#include <vector>
#include <random>
#include <array>

int main()
{
  std::array<double, 3> intervals = {0.0, 0.5, 1.0};
  std::array<double, 2> densities = {0.3, 0.5};

  std::piecewise_constant_distribution<> dist(
    intervals.begin(),
    intervals.end(),
    densities.begin()
  );

  std::vector<double> xs = dist.intervals();
  for (double x : xs) {
    std::cout << x << std::endl;
  }
}

出力

0
0.5
1
