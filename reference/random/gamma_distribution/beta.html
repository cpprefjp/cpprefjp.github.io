beta (C++11)

result_type beta() const;

概要

尺度母数(scale parameter)を取得する。

戻り値

構築時に設定された、尺度母数の値を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::gamma_distribution<> dist(1.0, 1.0);

  double beta = dist.beta();
  std::cout << beta << std::endl;
}

出力

1.0
