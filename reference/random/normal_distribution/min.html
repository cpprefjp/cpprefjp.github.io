min (C++11)

result_type min() const;

概要

生成する範囲の最小値を取得する。

戻り値

値の範囲の最小値を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::normal_distribution<> dist(0.0, 1.0);

  double min_val = dist.min();
  std::cout << min_val << std::endl;
}

出力例

-inf

言語

備考

Boost.Randomとlibc++(Clang)の実装では、-numeric_limits::infinity()を返す。

GCC 4.8.1時点でのlibstdc++の実装では、numeric_limits::min()を返す。これはつまり0.0を意味するが、実際には0.0未満の値を生成するため、これはバグであるBug 58098 - wrong return value of normal_distribution::min()。この問題は、GCC 4.8.2で修正された。
