max (C++11)

result_type max() const;

概要

生成する範囲の最大値を取得する。

戻り値

値の範囲の最大値を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::cauchy_distribution<> dist(0.0, 1.0);

  double max_val = dist.max();
  std::cout << max_val << std::endl;
}

出力例

1.79769e+308
