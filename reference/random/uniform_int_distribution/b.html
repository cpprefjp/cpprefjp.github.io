b (C++11)

result_type b() const;

概要

生成する範囲の最大値を取得する。

戻り値

構築時に設定された、値の範囲の最大値を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::uniform_int_distribution<> dist(0, 3);

  int b = dist.b();
  std::cout << b << std::endl;
}

出力

3
