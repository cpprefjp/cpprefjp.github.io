b (C++11)

RealType b() const;

概要

尺度パラメータを取得する。

戻り値

構築時に設定された、尺度パラメータを返す。

例

#include <iostream>
#include <random>

int main()
{
  std::extreme_value_distribution<> dist(0.0, 1.0);

  double b = dist.b();
  std::cout << b << std::endl;
}

出力

1
