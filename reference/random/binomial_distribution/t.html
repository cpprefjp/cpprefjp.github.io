t (C++11)

IntType t() const;

概要

試行回数tを取得する。

戻り値

構築時に設定された、試行回数tを返す。

例

#include <iostream>
#include <random>

int main()
{
  std::binomial_distribution<> dist(3, 0.5);

  int t = dist.t();
  std::cout << t << std::endl;
}

出力

3
