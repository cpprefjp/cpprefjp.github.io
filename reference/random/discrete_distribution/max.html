max (C++11)

result_type max() const;

概要

最大値を取得する。

戻り値

最大のインデックスである「確率列の要素数 - 1」を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::discrete_distribution<> dist = {0.1, 0.2, 0.3};

  int max_value = dist.max();
  std::cout << max_value << std::endl;
}

出力

2
