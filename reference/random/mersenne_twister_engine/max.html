max (C++11)

static constexpr result_type max();

概要

生成する値の最大値を取得する。

戻り値

2^w - 1を返す。

計算量

定数時間

例

#include <iostream>
#include <random>

int main()
{
  std::uint32_t max_value = std::mt19937::max();
  std::cout << max_value << std::endl;
}
