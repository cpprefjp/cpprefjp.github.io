p (C++11)

double p() const;

概要

確率pを取得する。

戻り値

構築時に設定された、trueを生成する確率pを返す。

例

#include <iostream>
#include <random>

int main()
{
  std::bernoulli_distribution dist(0.3);

  double p = dist.p();
  std::cout << p << std::endl;
}

出力

0.3
