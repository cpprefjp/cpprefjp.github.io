min (C++11)

result_type min() const;

概要

生成する範囲の最小値を取得する。

戻り値

falseを返す。

例

#include <iostream>
#include <random>

int main()
{
  std::bernoulli_distribution dist(0.5);

  bool min_value = dist.min();
  std::cout << std::boolalpha << min_value << std::endl;
}

出力

false
