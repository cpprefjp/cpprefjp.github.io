n (C++11)

result_type n() const;

概要

自由度を取得する。

戻り値

構築時に設定された、自由度を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::student_t_distribution<> dist(1.0);

  double n = dist.n();
  std::cout << n << std::endl;
}

出力

1
