operator() (C++11)

template <class URNG>
result_type operator()(URNG& g);

template <class URNG>
result_type operator()(URNG& g, const param_type& parm);

概要

指定されたパラメータに基いて、乱数生成を行う。

パラメータを受け取るバージョンは、コンストラクタで設定されたパラメータの代わりに、paramを乱数生成のパラメータとして使用する。

戻り値

指定されたパラメータに基いて、分布したランダムな値を生成して返す。

計算量

償却定数時間(g()の呼び出し回数)

例

#include <iostream>
#include <random>

int main()
{
  std::random_device seed_gen;
  std::default_random_engine engine(seed_gen());

  {
    std::poisson_distribution<> dist(1.0);

    for (int i = 0; i < 5; ++i) {
      int result = dist(engine);
      std::cout << result << std::endl;
    }
  }

  // パラメータを渡すバージョン
  std::cout << std::endl;
  {
    typedef std::poisson_distribution<> dist_type;
    dist_type dist;

    for (int i = 0; i < 5; ++i) {
      dist_type::param_type param(1.0);
      int result = dist(engine, param);
      std::cout << result << std::endl;
    }
  }
}
