m (C++11)

result_type m() const;

概要

生成する範囲の平均値を取得する。

戻り値

構築時に設定された、値の範囲の平均値を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::lognormal_distribution<> dist(0.0, 1.0);

  double m = dist.m();
  std::cout << m << std::endl;
}

出力

0
