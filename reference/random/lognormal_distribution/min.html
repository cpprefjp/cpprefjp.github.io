min (C++11)

result_type min() const;

概要

生成する範囲の最小値を取得する。

戻り値

値の範囲の最小値である0を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::lognormal_distribution<> dist(0.0, 1.0);

  double min_val = dist.min();
  std::cout << min_val << std::endl;
}

出力例

0
