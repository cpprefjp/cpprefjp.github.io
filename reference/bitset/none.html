none

bool none() const noexcept;

概要

全てのビットが0になっているかを判定する。

戻り値

全てのビットが0になっていればtrue、そうでなければfalseを返す。
この関数は、以下のプログラムと同じ動作をする：

count() == 0;return

例

#include <cassert>
#include <bitset>

int main()
{
  {
    std::bitset<4> bs("0000");

    bool result = bs.none();
    assert(result);
  }
  {
    std::bitset<4> bs("0010");

    bool result = bs.none();
    assert(!result);
  }
}
