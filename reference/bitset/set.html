set

bitset<N>& set();                            // (1) C++03
bitset<N>& set() noexcept;                   // (1) C++11

bitset<N>& set(size_t pos, bool val = true); // (2)

概要

任意の位置のビットを設定する。

要件

  • (2): pos < size()であること。

効果

  • (1): 全ビットを1にする。
  • (2): valの値がtrueであれば1、そうでなけば0を、pos番目のビット値として設定する。

戻り値

*this

例外

  • (1): 投げない。
  • (2): pos >= size()の場合、out_of_range例外を送出する。

例

#include <iostream>
#include <bitset>

int main()
{
  {
    std::bitset<4> bs;
    bs.set();

    std::cout << bs << std::endl;
  }

  {
    std::bitset<4> bs;
    bs.set(0).set(2);

    std::cout << bs << std::endl;
  }
}

出力

1111
0101
