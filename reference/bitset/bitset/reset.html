std::bitset::reset

bitset<N>& reset();           // (1) C++03
bitset<N>& reset() noexcept;  // (1) C++11

bitset<N>& reset(size_t pos); // (2)

概要

任意の位置のビットを0にする。

要件

  • (2): pos < size()であること。

効果

  • (1): 全ビットを0にする。
  • (2): pos番目のビットを0にする。

戻り値

*this

例外

  • (1): 投げない。
  • (2): pos >= size()の場合、out_of_range例外を送出する。

例

#include <iostream>
#include <bitset>

int main()
{
  {
    std::bitset<4> bs("1010");
    bs.reset();

    std::cout << bs << std::endl;
  }

  {
    std::bitset<4> bs("1010");
    bs.reset(1);

    std::cout << bs << std::endl;
  }
}

出力

0000
1000
