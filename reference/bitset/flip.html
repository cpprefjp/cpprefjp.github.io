flip

bitset<N>& flip();           // (1) C++03
bitset<N>& flip() noexcept;  // (1) C++11

bitset<N>& flip(size_t pos); // (2)

概要

ビットを反転させる。

要件

  • (2): pos < size()であること。

効果

  • (1): 全ビットを反転させる
  • (2): pos番目のビットを反転させる

戻り値

*this

例外

  • (1): 投げない。
  • (2): pos >= size()の場合、out_of_range例外を送出する。

例

#include <iostream>
#include <bitset>

int main()
{
  {
    std::bitset<4> bs("0011");
    bs.flip();

    std::cout << bs << std::endl;
  }

  {
    std::bitset<4> bs("0011");
    bs.flip(0);

    std::cout << bs << std::endl;
  }
}

出力

1100
0010
