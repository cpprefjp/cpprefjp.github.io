all (C++11)

bool all() const noexcept;

概要

全てのビットが1になっているかを判定する。

戻り値

全てのビットが1になっていればtrue、そうでなければfalseを返す。
この関数は、以下のプログラムと同じ動作をする：

count() == size();return

例

#include <cassert>
#include <bitset>

int main()
{
  {
    std::bitset<4> bs("1111");

    bool result = bs.all();
    assert(result);
  }
  {
    std::bitset<4> bs("0010");

    bool result = bs.all();
    assert(!result);
  }
}
