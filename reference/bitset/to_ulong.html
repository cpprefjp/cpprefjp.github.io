to_ulong

unsigned long to_ulong() const;

概要

unsigned long型に変換する。

戻り値

ビット列をunsigned long型に変換して返す。

例外

unsigned long型に変換した結果としてオーバーフローした場合、overflow_error例外を送出する。

例

#include <iostream>
#include <bitset>

int main()
{
  std::bitset<4> bs("1011");

  unsigned long result = bs.to_ulong();
  std::cout << result << std::endl;
}

出力

11
