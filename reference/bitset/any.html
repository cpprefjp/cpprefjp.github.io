any

bool any() const;          // C++03
bool any() const noexcept; // C++11

概要

いずれかのビットが1になっているかを判定する。

戻り値

いずれかのビットが1になっていればtrue、そうでなければfalseを返す。
この関数は、以下のプログラムと同じ動作をする：

count() != 0;return

例

#include <cassert>
#include <bitset>

int main()
{
  {
    std::bitset<4> bs("1011");

    bool result = bs.any();
    assert(result);
  }
  {
    std::bitset<4> bs("0000");

    bool result = bs.any();
    assert(!result);
  }
}
