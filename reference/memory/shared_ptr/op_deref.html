operator* (C++11)

T& operator*() const noexcept;

概要

ポインタを間接参照する。

要件

get() != nullptr

戻り値

*get()

備考

型Tがvoidの場合、この関数が定義されるかどうかは未規定。定義される場合、その戻り値は未規定。

例

#include <iostream>
#include <memory>

int main()
{
  std::shared_ptr<int> p(new int(3));

  int& r = *p;
  std::cout << r << std::endl;
}

出力

3
