operator bool (C++11)

explicit operator bool() const noexcept;

概要

有効なリソースを所有しているかを判定する。

戻り値

get() != nullptr

例

#include <iostream>
#include <memory>

int main()
{
  std::shared_ptr<int> p(new int(3));

  if (p) {
    std::cout << "p has resource" << std::endl;
  }
  else {
    std::cout << "p doesn't have resource" << std::endl;
  }
}

出力

p has resource

バージョン

言語

  • C++11

処理系

  • GCC: 4.4.7
  • Clang libc++, C++11 mode: 3.0
  • ICC: ?
  • Visual C++: 9.0 (TR1), 10.0, 11.0, 12.0
    • Visual C++ 11.0までは、コンパイラがexplicit operator boolに対応していないため、不透明な型へのポインタ型への変換演算子関数として実装されている。
