atomic_store (C++11)

namespace std {
  template <class T>
  void atomic_store(shared_ptr<T>* p, shared_ptr<T> r);
}

概要

shared_ptrオブジェクトに、アトミックに値を書き込む。

要件

p != nullptrであること。

効果

atomic_store_explicit(p, r, memory_order_seq_cst)

戻り値

なし

例外

投げない
