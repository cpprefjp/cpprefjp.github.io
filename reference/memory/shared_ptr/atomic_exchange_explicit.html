概要

メモリオーダーを指定して、shared_ptrオブジェクトを、アトミックに入れ替える。

要件

p != nullptrであること。

効果

p->swap(r)相当のことを、アトミックに実行する。

戻り値

変更前の*pを返す。

例外

投げない
