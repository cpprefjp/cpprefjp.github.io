reset (C++11)

void reset() noexcept;

概要

監視対象をクリアする。

効果

weak_ptr().swap(*this)と同等の効果を持つ。

戻り値

なし

例

#include <cassert>
#include <memory>

int main()
{
  // wpはspを監視する
  std::shared_ptr<int> sp(new int(3));
  std::weak_ptr<int> wp = sp;

  // spの監視をやめる
  wp.reset();

  assert(wp.use_count() == 0);
}
