uninitialized_fill

namespace std {
  template <class ForwardIterator, class T>
  void uninitialized_fill(ForwardIterator first, ForwardIterator last,
                          const T& x);
}

概要

未初期化領域の範囲を、指定された値で配置newする。

未初期化入力範囲[first, last)を初期化しつつ値xで埋める。

効果

iterator_traits<ForwardIterator>::value_type(x);for (; first != last; ++first)
  ::new (static_cast<void*>(&*first))
    typename

戻り値

なし
