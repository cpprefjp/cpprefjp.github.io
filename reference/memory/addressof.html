addressof (C++11)

namespace std {
  template <class T>
  T* addressof(T& r) noexcept;
}

概要

変数のアドレスを必ず取得する。

operator&() をオーバーロードしたクラスであっても、正しくそのオブジェクトのアドレスが欲しいという要求がある。 addressof()関数は、operator&() がオーバーロードされていても、変数のアドレスを取得できる。

戻り値

変数rのアドレスを返す。

例外

投げない
