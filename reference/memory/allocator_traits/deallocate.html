deallocate (C++11)

static void deallocate(Alloc& a, pointer p, size_type n);

概要

メモリを解放する。

効果

a.deallocate(p, n)

例外

投げない

例

#include <memory>

int main()
{
  std::allocator<int> alloc;
  using traits = std::allocator_traits<decltype(alloc)>;

  // 10要素のint領域を確保する
  std::size_t n = 10;
  int* p = traits::allocate(alloc, n);

  // 確保したメモリを解放する
  traits::deallocate(alloc, p, n);
}
