max_size (C++11)

static size_type max_size(Alloc& a);          // C++11
static size_type max_size(Alloc& a) noexcept; // C++14

概要

一度に確保可能なメモリの最大サイズを取得する。

戻り値

a.max_size()という式が有効ならその戻り値を返し、そうでなければデフォルト実装としてnumeric_limits<size_type>::max()の戻り値を返す。

例

#include <iostream>
#include <memory>

int main()
{
  std::allocator<int> alloc;
  using traits = std::allocator_traits<decltype(alloc)>;

  std::cout << traits::max_size(alloc) << std::endl;
}

出力例

4611686018427387903
