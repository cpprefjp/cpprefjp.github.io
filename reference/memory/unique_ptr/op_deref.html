operator* (C++11)

add_lvalue_reference<T>::type operator*() const;typename

概要

ポインタを間接参照する。

要件

get() != nullptr

戻り値

*get()

例

#include <iostream>
#include <memory>

int main()
{
  std::unique_ptr<int> p(new int(3));

  int& r = *p;
  std::cout << r << std::endl;
}

出力

3
