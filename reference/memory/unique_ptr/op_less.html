概要

unique_ptrにおいて、左辺が右辺より小さいかを判定する。

比較対象は、unique_ptrが指す値ではなく、unique_ptrが保持するポインタ値。

戻り値

  • (1) : std::common_type<unique_ptr<T1, D1>::pointer, unique_ptr<T2, D2>::pointer>::typeを、aとbが持つポインタの共通の型CTとし、std::less<CT>(a.get(), b.[get()](./get.md))`で比較した結果を返す。

  • (2) : std::less<unique_ptr<T, D>::pointer>()(x.get(), nullptr)で比較した結果を返す。

  • (3) : std::less<unique_ptr<T, D>::pointer>()(nullptr, x.get())で比較した結果を返す。

例

#include <iostream>
#include <memory>

int main()
{
  std::cout << std::boolalpha;

  std::unique_ptr<int> p1(new int(3));
  std::unique_ptr<int> p2(new int(3));

  bool r1 = p1 < p2;
  std::cout << r1 << std::endl;

  bool r2 = p1 < nullptr;
  std::cout << r2 << std::endl;

  bool r3 = nullptr < p1;
  std::cout << r3 << std::endl;
}

false
