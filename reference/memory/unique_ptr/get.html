get (C++11)

pointer get() const noexcept;

概要

リソースを取得する。

戻り値

保持しているポインタを返す。

備考

リソースの所有権は*thisが持っているので、この関数によって返されたポインタに対して、リソース解放をしてはならない。

例

#include <iostream>
#include <memory>

int main()
{
  std::unique_ptr<int> p(new int(3));

  // リソースを取得
  int* raw_pointer = p.get();
  std::cout << *raw_pointer << std::endl;
}

出力

3
