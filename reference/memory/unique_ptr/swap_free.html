swap (フリー関数版)(C++11)

namespace std {
  template <class T, class D>
  void swap(unique_ptr<T, D>& a, unique_ptr<T, D>& b) noexcept;
}

概要

2つのunique_ptrオブジェクトを入れ替える。

効果

a.swap(b);

戻り値

なし

例

#include <iostream>
#include <memory>

int main()
{
  std::unique_ptr<int> a(new int(3));
  std::unique_ptr<int> b(new int(1));

  // aとbを入れ替える
  std::swap(a, b);

  std::cout << *a << std::endl;
  std::cout << *b << std::endl;
}

出力

1
3
