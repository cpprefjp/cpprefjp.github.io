get_deleter (C++11)

deleter_type& get_deleter() noexcept;
const deleter_type& get_deleter() const noexcept;

概要

デリータを取得する。

戻り値

保持しているデリータオブジェクトへの参照を返す。

例

#include <iostream>
#include <memory>

int main()
{
  std::unique_ptr<int> p(new int(3));

  // デリータを取得
  std::default_delete<int>& f = p.get_deleter();
}
