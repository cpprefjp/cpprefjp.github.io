release (C++11)

pointer release() noexcept;

概要

リソースの所有権を放棄する。

効果

*thisが保持しているリソースの所有権を放棄する。リソースを解放するのではなく、解放する責任を放棄する。

この関数を呼び出したあと、このクラスのデストラクタでは、現在保持しているリソースを解放しなくなる。

事後条件

この関数を呼び出したあと、get()メンバ関数はnullptrを返す。

戻り値

放棄したリソースを返す。

例

#include <memory>

int main()
{
  std::unique_ptr<int> p(new int(3));

  // pは、保持しているリソースの所有権を放棄する
  // resource変数が所有権を持つようになる
  int* resource = p.release();

  // 所有権を持つresourceがリソースを解放する
  delete resource;

  // pのデストラクタでは、リソースが解放されない
}
