clear

void clear() noexcept;

概要

multimap コンテナ内の全ての要素を削除する。それぞれのデストラクタが呼ばれ、コンテナから削除される。size() は 0 になる。

計算量

線形時間

例

#include <iostream>
#include <map>

using namespace std;

int main() 
{
  multimap<int, char> c;
  c.insert( std::make_pair(3,'C'));
  c.insert( std::make_pair(4,'D'));
  c.insert( std::make_pair(1,'A'));
  c.insert( std::make_pair(2,'B'));

  std::cout << c.size() << endl;

  c.clear();

  std::cout << c.size() << endl;

  return 0;
}

出力

4
0
