cend (C++11)

// since C++11
const_iterator cend() const noexcept;

概要

map コンテナの最後の要素の次を参照するイテレータを返す。

戻り値

コンテナの最後の要素の次を参照するイテレータ。 const_iterator はいずれもメンバ型である。map クラステンプレートにおいて、これらは双方向イテレータである。

計算量

定数時間

例

#include <iostream>
#include <map>

using namespace std;

int main() 
{
  map<int, char> c;
  c[3] = 'C';
  c[7] = 'G';
  c[8] = 'H';
  c[4] = 'D';
  c[5] = 'E';
  c[1] = 'A';
  c[2] = 'B';
  c[6] = 'F';

  for( auto i = c.begin(); i != c.end() ; ++i ) {
    cout << i->first << " " << i->second << "\n";
  }

  return 0;
}

出力

1 A
2 B
3 C
4 D
5 E
6 F
7 G
8 H
