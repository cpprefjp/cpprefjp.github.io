resize

// C++03まで
void resize(size_type sz, const T& c = T());

// C++11から
void resize(size_type sz);
void resize(size_type sz, const T& c);

概要

要素数を変更する

効果

コンテナのサイズをszに変更する。 もしszが現在のコンテナのsize()より小さい場合、後方のsize()- sz個の要素を削除する。 もしszが現在のコンテナのsize()より大きい場合、sz -size()個だけオブジェクトcのコピーを追加する。1引数版の場合、値初期化されたオブジェクトのコピーが追加される。

戻り値

なし
