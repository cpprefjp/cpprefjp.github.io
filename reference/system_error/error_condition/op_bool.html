explicit operator bool (C++11)

explicit operator bool() const noexcept;

概要

error_conditionオブジェクトがエラー状態であるかを判定する。error_conditionクラスのデフォルトエラー値である0が正常と見なされる。trueの場合はエラーであることを意味し、falseの場合は正常を意味する。

戻り値

value()!= 0

例外

投げない

if (ec) {
