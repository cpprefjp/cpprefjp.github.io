operator- (単項)

namespace std {
  template <class T>
  complex<T> operator-(const complex<T>& lhs);
}

概要

単項 - 演算（符号を反転した複素数値を得る）

戻り値

complex<T>(-lhs.real(), -lhs.imag())

例

iostream>
#include <complex>

int main()
{
  std::complex<float> c(1.0, 2.0);
  std::cout << '-' << c << " = " << -c << std::endl;
}#include <

出力

-(1,2) = (-1,-2)
