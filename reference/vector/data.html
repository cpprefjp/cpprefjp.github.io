data (C++11)

T* data() noexcept;
const T* data() const noexcept;

効果

配列の先頭へのポインタを返す。

vectorが空の場合であっても、この関数の呼び出し自体は問題なく行える。ただし、その戻り値については規定されていないため、間接参照を行うと未定義動作になる。

戻り値

[data(), data() + size()) が適正な範囲になるようなポインタ。

空ではないvectorに対してはdata() == &front()となる。
