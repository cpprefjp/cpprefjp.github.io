uncaught_exception

namespace std {
  // C++03
  bool uncaught_exception() throw();

  // C++11
  bool uncaught_exception() noexcept;
}

概要

キャッチされていない例外があるかどうかを判定する。

戻り値

例外オブジェクトが生成され、スローされてからキャッチするまでの間にtrueを返す。 また、明示的にterminate()を呼び出し、実際に呼び出されるまでの間にtrueを返す。

具体的には、tryブロック中で作られたオブジェクトのデストラクタや、スタック巻き戻し(unwind)中のデストラクタ、terminate()の場合は生存している全てのオブジェクトのデストラクタでtrueになる。

例外

投げない
