make_exception_ptr (C++11)

namespace std {
  template <class E>
  exception_ptr make_exception_ptr(E e) noexcept;
}

概要

引数の例外オブジェクトを元にexception_ptrを生成する。

効果

try { throw e; } catch(...) { return current_exception(); }

戻り値

例外

{
  try {
    throw e;
  }
  catch (...) {
    return current_exception();
  }
}
