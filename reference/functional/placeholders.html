placeholders (C++11)

unspecified _1;
  extern unspecified _2;
  …
  extern unspecified _N;
}}namespace std {
namespace placeholders {
  extern

概要

_1、_2、…、_Nは、bind()で使用するプレースホルダーオブジェクトである。

それぞれ、bind()によって返された関数オブジェクトに指定する「第1引数」「第2引数」…「第N引数」であることを意味する。

備考

プレースホルダーの数(Nの値)は、10以上実装することが推奨される。
