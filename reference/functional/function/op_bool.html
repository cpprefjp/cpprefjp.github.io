operator bool (C++11)

explicit operator bool() const noexcept;

概要

関数呼び出しが可能か調べる。

戻り値

呼び出す関数を持っていればtrue、そうでなければfalseを返す。

例

#include <iostream>
#include <functional>

int ident(int x) { return x; }

int main()
{
  std::function<int(int)> f = ident;

  if (f) {
    std::cout << "not empty" << std::endl;
  }
  else {
    std::cout << "empty" << std::endl;
  }
}

出力

not empty
