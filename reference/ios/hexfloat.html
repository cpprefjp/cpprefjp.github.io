std::hexfloat(C++11)

namespace std {
  ios_base& hexfloat(ios_base& str);
}

概要

浮動小数点数を十六進法で出力することを指示するマニピュレータ。

printf()関数の%a／%A相当。

効果

str.setf(ios_base::fixed | ios_base::scientific, ios_base::floatfield)を実行する。

戻り値

実引数のstrオブジェクト。

例

defaultfloatを参照。
