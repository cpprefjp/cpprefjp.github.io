boolalpha

namespace std {
  ios_base& boolalpha(ios_base& str);
}

概要

bool値を文字列として入出力することを指示するマニピュレータ。

効果

str.setf(std::ios_base::boolalpha)を実行する。

戻り値

実引数のstrオブジェクト。

例

#include <iostream>

int main()
{
  std::cout << std::boolalpha << true << std::endl;
}

出力

true
