dec

namespace std {
  ios_base& dec(ios_base& str);
}

概要

整数を十進法で出力することを指示するマニピュレータ。

効果

str.setf(ios_base::dec, ios_base::basefield)を実行する。

戻り値

実引数のstrオブジェクト。

例

#include <iostream>

int main()
{
  for (int i = 0; i < 20; ++i) {
    std::cout << std::dec << i << '\t';
    std::cout << std::oct << i << '\t';
    std::cout << std::hex << i << std::endl;
  }
}
