bucket_count (C++11)

size_type bucket_count() const noexcept;

概要

コンテナの現在のバケット数を返す。

戻り値

コンテナの現在のバケット数

例外

投げない。

計算量

定数。

例

iostream>
#include <string>
#include <unordered_map>

int main()
{
  const std::unordered_multimap<std::string, int> um = {
    {"1st", 1},
    {"2nd", 2},
    {"3rd", 3},
    {"4th", 4},
    {"5th", 5}
  };

  std::cout << "bucket count is " << um.bucket_count() << std::endl;
}#include <

出力例

bucket count is 11
