max_size (C++11)

size_type max_size() const noexcept;

概要

格納可能な最大の要素数を取得する。

戻り値

このコンテナに格納可能な最大の要素数

例外

投げない。

計算量

定数

例

iostream>
#include <string>
#include <unordered_map>

int main()
{
  std::unordered_multimap<std::string, int> um;
  decltype(um)::size_type s = um.max_size();

  std::cout << s << std::endl;
}#include <

出力例

576460752303423487

備考

libstdc++ では、コンテナのアロケータ allocator_type にメンバ関数 max_size が無いとコンパイル時エラーが発生する。 （allocator_type::max_size() を std::allocator_traits::max_size 経由ではなく直接使用しているため。）
