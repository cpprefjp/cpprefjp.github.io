empty (C++11)

bool empty() const noexcept;

概要

コンテナが空かどうかを判定する。

戻り値

コンテナが空であれば true、そうでなければ false を返す。

例外

投げない。

計算量

定数

例

iostream>
#include <string>
#include <unordered_map>

int main()
{
  std::cout << std::boolalpha;

  std::unordered_map<std::string, int> um;

  // 空
  std::cout << um.empty() << std::endl;

  um.emplace("1st", 1);

  // 空ではない
  std::cout << um.empty() << std::endl;

  um.clear();

  // 空
  std::cout << um.empty() << std::endl;
