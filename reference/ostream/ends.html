概要

ヌル文字を出力する。

効果

  1. os.put(CharT())を呼び出す。

戻り値

os

実装例

namespace std {
  template<class CharT, class Traits>
  basic_ostream<CharT, Traits>& endl(basic_ostream<CharT, Traits>& os) {
    return os.put(CharT());
  }
}
