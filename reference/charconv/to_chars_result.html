std::to_chars_result(C++17)

namespace std {
  struct to_chars_result {
    char* ptr;
    errc ec;
  };
}

概要

to_charsの戻り値となる変換の成否と追加情報を表現するクラス。

メンバ変数

名前 説明 対応バージョン
ptr 成功した場合、書き込まれた最後の文字の次の位置を指すポインタ。 C++17
ec 成功ならec == errc{}、失敗ならばerrc::value_too_largeを表すerrcの値。 C++17

詳細はto_charsの戻り値の項を参照のこと。
