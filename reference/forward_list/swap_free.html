swap(非メンバ関数) (C++11)

namespace std {
  template <class T, class Allocator>
  void swap(forward_list<T, Allocator>& x, forward_list<T, Allocator>& y);
}

概要

2つのforward_listオブジェクトを入れ替える。

効果

x.swap(y)

戻り値

なし
