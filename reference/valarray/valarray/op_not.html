operator!

valarray<bool> operator!() const;

概要

各要素の論理NOTをとる。

戻り値

以下のコードと同等のことを行う：

valarray<bool> result(size());
for (size_t i = 0; i < size(); ++i) {
  result[i] = !(*this)[i];
}
return result;

例

#include <cassert>
#include <valarray>

int main()
{
  const std::valarray<bool> v = {true, false, true};

  std::valarray<bool> result = !v;
  assert(!result[0]);
  assert(result[1]);
  assert(!result[2]);
}
