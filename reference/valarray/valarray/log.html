log

namespace std {
  template <class T>
  valarray<T> log(const valarray<T>& v);
}

概要

自然対数（ネイピア数 e を底とした対数）を得る。log は logarithm（対数）、あるいは、logarithmic function（対数関数）の略。

戻り値

以下のコードと同等のことを行う：

apply(static_cast<T(*)(T)>(std::log));return v.

{

  bool first = true;

  // 範囲for文で全要素を走査する
  for (const T& x : v) {
    if (first) {
      first = false;
    }
    else {
      std::cout << ',';
    }
    std::cout << x;
  }
  std::cout << "}" << std::endl;
}
