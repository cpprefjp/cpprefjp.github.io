apply

valarray<T> apply(T func(T)) const;
valarray<T> apply(T func(const T&)) const;

概要

各要素に任意の関数を適用する。

戻り値

*thisの全ての要素に関数funcを適用したvalarrayオブジェクトを新たに作って返す。

返されるvalarrayオブジェクトは、*thisと同じ要素数、同じ要素型を持つ。

例

apply([](int x) { return x + 1; });
