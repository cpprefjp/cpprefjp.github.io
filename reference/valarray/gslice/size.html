size

valarray<size_t> size() const;

概要

スライスを生成する要素数群を取得する。

戻り値

スライスを生成する要素数群。

例

#include <valarray>
#include <iostream>

auto main()
  -> int
{
  constexpr auto             start   = 3;
  std::valarray<std::size_t> lengths = {  3, 4 };
  std::valarray<std::size_t> strides = { 10, 3 };

  std::gslice gs( start, lengths, strides );

  for ( auto v : gs.size() )
    std::cout << v << "\n";
  std::cout << std::flush;
}

出力

3
4
