start

size_t start() const;

概要

スライスを生成する初期位置を取得する。

戻り値

スライスを生成する初期位置。

例

#include <valarray>
#include <iostream>

auto main()
  -> int
{
  constexpr auto             start   = 3;
  std::valarray<std::size_t> lengths = {  3, 4 };
  std::valarray<std::size_t> strides = { 10, 3 };

  std::gslice gs( start, lengths, strides );

  std::cout << gs.start();
}

出力

3
