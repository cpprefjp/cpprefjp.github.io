operator>>=

void operator>>=(const valarray<T>& xs) const;

概要

右シフトの複合代入を行う。

効果

元となるvalarrayオブジェクトから参照によって抽出した各要素を、xsの各要素の値だけ左シフトする。

戻り値

なし

備考

valarrayから抽出した要素数とxsの要素数が異なる場合、その挙動は未定義。

例

#include <iostream>
#include <valarray>
#include <bitset>
#include <numeric> // accumulate
#include <functional> // multiplies

std::size_t product(const std::valarray<std::size_t>& v)
{
  return std::accumulate(
           std::begin(v),
           std::end(v),
           1u,
           std::multiplies<std::size_t>());
}

int main()
{
  std::valarray<int> v = {
    0b01010001,
    0b10100000,
    0b01011000
  };

  const std::size_t start = 0u;  // 開始位置
  const std::valarray<std::size_t> lengths = {3u}; // 要素数
  const std::valarray<std::size_t> strides = {1u}; // 何個置きに処理するか

  // 抽出した要素を4ビット右シフトする
  // lengthを掛け合わせたものがgsliceで抽出した配列の長さ
  v[std::gslice(start, lengths, strides)] >>= std::valarray<int>(4, product(lengths));

  for (int x : v) {
    std::cout << std::bitset<8>(x).to_string() << std::endl;
  }
}

出力

00000101
00001010
00000101
