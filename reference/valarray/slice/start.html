start

size_t start() const;

概要

スライスを生成する初期位置を取得する。

戻り値

スライスを生成する初期位置。

例

#include <valarray>
#include <iostream>

auto main()
  -> int
{
  constexpr auto start  = 3;
  constexpr auto length = 5;
  constexpr auto stride = 7;

  std::slice s( start, length, stride );

  std::cout << s.start();
}

出力

3
