operator%=

void operator%=(const valarray<T>& xs) const;

概要

剰余算の複合代入を行う。

効果

元となるvalarrayオブジェクトから参照によって抽出した各要素に、xsの各要素を剰余算する。

戻り値

なし

備考

valarrayから抽出した要素数とxsの要素数が異なる場合、その挙動は未定義。

例

#include <iostream>
#include <valarray>

int main()
{
  std::valarray<int> v = {1, 2, 3, 4, 5, 6};
  std::valarray<bool> mask = {false, true, false, true, false, true};

  std::mask_array<int> result = v[mask];

  // 抽出した要素に3を剰余算する
  result %= std::valarray<int>(3, 3);

  for (int x : v) {
    std::cout << x << std::endl;
  }
}

出力

1
2
3
1
5
0
