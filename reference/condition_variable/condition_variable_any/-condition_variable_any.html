デストラクタ (C++11)

~condition_variable_any();

概要

condition_variable_anyオブジェクトの破棄を行う

要件

*thisを参照している全てのスレッドがブロッキングされていないこと。 つまり、デストラクタが呼び出される前に、*thisのcondition_variableオブジェクトを待機している全てのスレッドに起床通知を送らなければならない。

効果

条件変数を破棄する
