conditional (C++11)

namespace std {
  template <bool B, class T, class F>
  struct conditional {
    typedef … type;
  };
}

概要

コンパイル時条件式。
条件式がtrueかfalseかによって、使用する型を切り替える。

効果

conditionalは、条件式Bがtrueであれば型Tを、そうでなければ型Fを、メンバ型typeとして定義する。

例

#include <type_traits>

static_assert(std::is_same<std::conditional<true, int, char>::type, int>::value, "select int");
static_assert(std::is_same<std::conditional<false, int, char>::type, char>::value, "select char");

int main() {}
