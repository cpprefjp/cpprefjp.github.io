rank (C++11)

namespace std {
  template <class T>
  struct rank {
    typedef … type;
  };
}

概要

配列型の次元数を取得する。

効果

型Tが配列型である場合、配列の次元数となる整数値をメンバ定数valueの値として定義する。配列型でなければ0をメンバ定数valueの値として定義する。

例

#include <type_traits>

static_assert(std::rank<int>::value == 0, "int rank is 0");
static_assert(std::rank<int[3]>::value == 1, "int[3] rank is 1");
static_assert(std::rank<int[3][4]>::value == 2, "int[3][4] rank is 2");

int main() {}
