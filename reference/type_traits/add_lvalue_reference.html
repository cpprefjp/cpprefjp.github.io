add_lvalue_reference (C++11)

namespace std {
  template <class T>
  struct add_lvalue_reference {
    typedef … type;
  };
}

概要

型に左辺値参照を追加する。

効果

add_lvalue_referenceは、型Tの名前に&修飾を付加した型を、メンバ型typeとして定義する。
型Tが何らかのオブジェクト型もしくは関数型Uへの左辺値参照型である場合は、型U&をメンバ型typeとして定義する。
型Tが何らかのオブジェクト型もしくは関数型Uへの右辺値参照型である場合は、型U&をメンバ型typeとして定義する。

例

#include <type_traits>

static_assert(std::is_same<std::add_lvalue_reference<int>::type, int&>::value, "transform int to int&");
static_assert(std::is_same<std::add_lvalue_reference<int&>::type, int&>::value, "transform int& to int&");
static_assert(std::is_same<std::add_lvalue_reference<int&&>::type, int&>::value, "transform int&& to int&");

int main() {}
