add_pointer (C++11)

namespace std {
  template <class T>
  struct add_pointer {
    typedef … type;
  };
}

概要

型にポインタを追加する。

効果

add_pointerは、型remove_reference<T>::type*を、メンバ型typeとして定義する。

例

#include <type_traits>

static_assert(std::is_same<std::add_pointer<int>::type, int*>::value, "transform int to int*");
static_assert(std::is_same<std::add_pointer<int&>::type, int*>::value, "transform int& to int*");
static_assert(std::is_same<std::add_pointer<int*>::type, int**>::value, "transform int* to int**");
static_assert(std::is_same<std::add_pointer<int*&>::type, int**>::value, "transform int*& to int**");

int main() {}
