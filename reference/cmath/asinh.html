概要

算術型の逆双曲線正弦（アークハイパボリックサイン）を求める。

戻り値

引数 x の逆双曲線正弦を返す。

x が ±∞ だった場合 ±∞ を返す。

備考

例

#include <cmath>
#include <iostream>

int main() {
  std::cout << std::fixed;
  std::cout << "asinh(-1.0) = " << std::asinh(-1.0) << std::endl;
  std::cout << "asinh(0.0)  = " << std::asinh(0.0) << std::endl;
  std::cout << "asinh(1.0)  = " << std::asinh(1.0) << std::endl;
}

出力

asinh(-1.0) = -0.881374
asinh(0.0)  = 0.000000
asinh(1.0)  = 0.881374
