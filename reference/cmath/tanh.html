概要

算術型の双曲線正接（ハイパボリックタンジェント）を求める。

戻り値

引数 x の双曲線正接を返す。

備考

例

#include <cmath>
#include <iostream>

int main() {
  std::cout << std::fixed;
  std::cout << "tanh(-1.0) = " << std::tanh(-1.0) << std::endl;
  std::cout << "tanh(0.0)  = " << std::tanh(0.0) << std::endl;
  std::cout << "tanh(1.0)  = " << std::tanh(1.0) << std::endl;
}

出力

tanh(-1.0) = -0.761594
tanh(0.0)  = 0.000000
tanh(1.0)  = 0.761594
