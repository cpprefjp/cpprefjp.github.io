概要

2 を底とする指数関数を求める。

戻り値

2 の x 乗を返す。

x が +∞ だった場合 +∞ を返す。

x が -∞ だった場合 0 を返す。

備考

例

#include <cmath>
#include <limits>
#include <iostream>

int main() {
  std::cout << std::fixed;
  std::cout << "exp2(0.0) = " << std::exp2(0.0) << std::endl;
  std::cout << "exp2(1.0) = " << std::exp2(1.0) << std::endl;
  std::cout << "exp2(+∞) = " << std::exp2(std::numeric_limits<double>::infinity()) << std::endl;
  std::cout << "exp2(-∞) = " << std::exp2(-std::numeric_limits<double>::infinity()) << std::endl;
}

出力

exp2(0.0) = 1.000000
exp2(1.0) = 2.000000
exp2(+∞) = inf
exp2(-∞) = 0.000000
