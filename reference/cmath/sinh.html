概要

算術型の双曲線正弦（ハイパボリックサイン）を求める。

戻り値

引数 x の双曲線正弦を返す。

備考

例

#include <cmath>
#include <iostream>

int main() {
  std::cout << std::fixed;
  std::cout << "sinh(-1.0) = " << std::sinh(-1.0) << std::endl;
  std::cout << "sinh(0.0)  = " << std::sinh(0.0) << std::endl;
  std::cout << "sinh(1.0)  = " << std::sinh(1.0) << std::endl;
}

出力

sinh(-1.0) = -1.175201
sinh(0.0)  = 0.000000
sinh(1.0)  = 1.175201
