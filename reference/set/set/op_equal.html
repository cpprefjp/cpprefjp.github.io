<set>

std::operator==

namespace std {
  template <class Key, class Compare, class Allocator>
  bool operator==(const set<Key,Compare,Allocator>& x, const set<Key,Compare,Allocator>& y);
}

概要

x が y と等しいかどうかの判定を行う。

戻り値

  • C++03 : x.size() == y.size() && equal(x.begin(), x.end(), y.begin());
  • C++14 : equal(x.begin(), x.end(), y.begin(), y.end());

計算量

size() に対して線形時間。ただし、xとyのサイズが異なる場合は定数時間。

例

#include <iostream>
#include <set>

int main()
{
  std::set<int> s1, s2;
  s1.insert(10);
  s1.insert(20);
  s1.insert(30);
  s2 = s1;

  std::cout << (s1 == s2) << std::endl;

  s2.insert(40);

  std::cout << (s1 == s2) << std::endl;
}

出力

1
0
