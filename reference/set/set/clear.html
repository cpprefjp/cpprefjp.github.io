function
<set>

std::set::clear

void clear() noexcept;

概要

set コンテナ内の全ての要素を削除する。それぞれのデストラクタが呼ばれ、コンテナから削除される。size() は 0 になる。

計算量

線形時間

例外

投げない

例

#include <iostream>
#include <set>

int main ()
{
  std::set<int> c;

  c.insert(10);
  c.insert(20);
  c.insert(30);

  std::cout << c.size() << std::endl;

  c.clear();

  std::cout << c.size() << std::endl;
}

出力

3
0
