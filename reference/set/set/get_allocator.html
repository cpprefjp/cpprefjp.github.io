get_allocator

allocator_type get_allocator() const noexcept;

概要

このコンテナで使用されているアロケータオブジェクトを取得する。

戻り値

このコンテナで使用されているアロケータオブジェクト

例外

投げない

計算量

定数時間

例

#include <iostream>
#include <set>
using namespace std;

int main()
{
  set<int> c;
  int * p;

  p = c.get_allocator().allocate(2);

  p[0] = 42;
  p[1] = 43;

  cout << p[0] << " " << p[1];

  c.get_allocator().deallocate(p, 2);

  return 0;
}

出力

42 43
