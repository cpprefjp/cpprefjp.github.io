end, cend

iterator end() noexcept;
const_iterator end() const noexcept;

// since C++11
const_iterator cend() const noexcept;

概要

set コンテナの最後の要素の次を参照するイテレータを返す。

戻り値

コンテナの最後の要素の次を参照するイテレータ。 iterator と const_iterator はいずれもメンバ型である。set クラステンプレートにおいて、これらは双方向イテレータである。

計算量

定数時間

例

#include <iostream>
#include <set>
using namespace std;

int main()
{
  set<int> c;
  c.insert(5);
  c.insert(2);
  c.insert(4);
  c.insert(1);
  c.insert(0);
  c.insert(0);
  c.insert(9);

  set<int>::iterator i = c.begin();
  while (i != c.end())
    cout << *i++ << " ";

  return 0;
}

0 1 2 4 5 9
