std::range_format(C++23)

namespace std {
  enum class range_format {
    disabled,
    map,
    set,
    sequence,
    string,
    debug_string
  };
}

概要

Rangeの書式種別を表す列挙型。

列挙値 説明
disabled 書式化ができないRangeであることを表す値
map std::mapやstd::flat_map向けの書式。{k1: v1, k2: v2}
set std::setやstd::flat_set向けの書式。{v1, v2}
sequence std::vectorやstd::array向けの書式。[v1, v2]
string 文字列向けの書式。abc
debug_string デバッグ文字列向けの書式。\nabc

備考

  • disableは値0であるため、bool型に変換して有効かどうかを判定するための使用できる
