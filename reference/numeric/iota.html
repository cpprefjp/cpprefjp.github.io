iota (C++11)

namespace std{
  template <class ForwardIterator, class T>
  void iota(ForwardIterator first, ForwardIterator last, T value);
}

概要

指定された値から始まる整数列を生成する。

パラメータ
first シーケンスの先頭
last シーケンスの終端
value 初期値

効果

全ての要素に対して、first から順番に *it = value; ++value; を行う

戻り値

なし

計算量

n 回のインクリメントと代入が行われる。

例

std::iota(s.begin(), s.end(), 'A');
  std::cout << s << std::endl;
