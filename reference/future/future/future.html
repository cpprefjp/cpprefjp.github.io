コンストラクタ (C++11)

future() noexcept;
future(future&& rhs) noexcept;
future(const future& rhs) = delete;

futureオブジェクトの構築

  • future() noexcept;
    デフォルトコンストラクタ。共有状態を持たない空のfutureオブジェクトを生成する。
    事後条件： [valid()](/reference/future/future/valid.md) == false
  • future(future&& rhs) noexcept;
    ムーブコンストラクタ。rhsオブジェクトが持つ共有状態を*thisに移動する。
    事後条件： valid()が、この関数実行前のrhs.valid()と等価になること。rhs.valid()== falseになること。
  • future(const future& rhs) = delete;
    コピーコンストラクタ。コピー不可。

例

#include <future>

int main()
{
  // デフォルト構築
  {
    std::future<int> f;
  }

  // ムーブ構築
  {
    std::promise<int> p;
    std::future<int> f = p.get_future();
  }
}
