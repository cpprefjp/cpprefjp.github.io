swap (フリー関数版)(C++11)

namespace std {
  template <class R, class... ArgTypes>
  void swap(packaged_task<R(ArgTypes...)>& x, packaged_task<R(ArgTypes...)>& y) noexcept;
}

概要

効果

戻り値

なし

例外

投げない
