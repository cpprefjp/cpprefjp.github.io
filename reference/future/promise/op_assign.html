代入演算子 (C++11)

promise& operator=(promise&& rhs) noexcept;
promise& operator=(const promise& rhs) = delete;

概要

  • promise& operator=(promise&& rhs) noexcept;
    ムーブ代入演算子。現在の共有状態を放棄し、promise(std::move(rhs)).swap(*this)する。
    戻り値： *this
  • promise& operator=(const promise& rhs) = delete;
    コピー代入演算子。コピー禁止。

例

#include <utility>
#include <future>

int main()
{
  std::promise<int> p1;
  std::promise<int> p2;
  p2 = std::move(p1);
}
