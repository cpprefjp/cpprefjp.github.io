operator-= (C++11)

time_point& operator-=(const duration& d);

概要

time_pointの時間を戻す

効果

d_ -= d ※ d_はtime_point内部で保持しているduration。変数名は説明用。

戻り値

*this

例

#include <iostream>
#include <chrono>

using namespace std::chrono;

int main()
{
  system_clock::time_point p = system_clock::now();

  p -= seconds(3);

  std::cout << p.time_since_epoch().count() << std::endl;
}

出力例

1314362390421481
