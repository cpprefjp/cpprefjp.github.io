概要

ミリ秒単位を表現するdurationの別名

例

#include <iostream>
#include <chrono>

int main()
{
  std::chrono::milliseconds ns1(30);
  std::chrono::milliseconds ns2(20);

  std::chrono::milliseconds result = ns1 + ns2;

  std::cout << result.count() << std::endl;
}

出力

50
