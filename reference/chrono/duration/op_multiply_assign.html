operator*= (C++11)

duration& operator*=(const rep& rhs);

概要

現在の値にrhsを掛ける

効果

rep_ *= rhs

戻り値

*this

例

#include <iostream>
#include <chrono>

using std::chrono::duration;
using std::micro;

int main()
{
  duration<int, micro> d(3);

  d *= 3;

  std::cout << d.count() << std::endl;
}

出力

9
