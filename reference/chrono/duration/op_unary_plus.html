operator+ (C++11)

constexpr duration operator+() const;

概要

正のdurationを生成する

戻り値

*this

例

#include <iostream>
#include <chrono>

using std::chrono::duration;
using std::nano;

int main()
{
  duration<int, nano> d1(2);

  duration<int, nano> d2 = +d1;

  std::cout << d1.count() << std::endl;
  std::cout << d2.count() << std::endl;
}

出力

2
2
