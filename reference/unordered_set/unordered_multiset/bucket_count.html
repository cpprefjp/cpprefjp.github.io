bucket_count (C++11)

size_type bucket_count() const noexcept;

概要

コンテナの現在のバケット数を返す。

戻り値

コンテナの現在のバケット数

例外

投げない。

計算量

定数。

例

iostream>
#include <unordered_set>

int main()
{
  std::unordered_multiset<int> um{ 1, 3, 5, 7, 9, 1, 3, 5, 7, 9, };

  std::cout << "bucket count is " << um.bucket_count() << std::endl;
}#include <

出力例

bucket count is 11
