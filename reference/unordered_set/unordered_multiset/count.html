count (C++11)

size_type count(const key_type& k) const;

概要

指定されたキーの要素数を数える。

戻り値

引数 k と等価なキーの要素数を返す。

計算量

平均的なケースでは定数（O(1)）だが、最悪のケースではコンテナの要素数 size() に比例（O(size())）。

例

iostream>
#include <unordered_set>
#include <algorithm>
#include <iterator>

int main()
{
  std::unordered_multiset<int> um{ 1, 3, 5, 7, 9, 1, 3, 5, 7, 9, };

  std::copy(um.begin(), um.end(), std::ostream_iterator<int>(std::cout, ", "));
  std::cout << std::endl;

  auto c1 = um.count(5);
  std::cout << "count of 5:" << c1 << std::endl;

  auto c2 = um.count(8);
  std::cout << "count of 8:" << c2 << std::endl;
}#include <

出力

9, 9, 7, 7, 5, 5, 3, 3, 1, 1,
count of 5:2
count of 8:0
