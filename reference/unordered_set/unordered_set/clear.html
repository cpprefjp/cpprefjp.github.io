clear (C++11)

void clear() noexcept;

概要

コンテナ内のすべての要素を削除する。

要件

なし。

効果

コンテナ内のすべての要素を削除する。

事後条件

empty() == true

戻り値

なし。

例外

投げない。

計算量

本関数呼び出し前のコンテナの要素数（size()）に比例

例

iostream>
#include <unordered_set>

int main()
{
  std::unordered_set<int> us{ 1, 2, 3, 4, 5, };

  std::cout << std::boolalpha;
  std::cout << us.empty() << std::endl;
  us.clear();
  std::cout << us.empty() << std::endl;
}#include <

出力

false
true
