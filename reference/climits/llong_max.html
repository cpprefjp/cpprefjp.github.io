LLONG_MAX (C++11)

implementation-defined#define LLONG_MAX

概要

long long の最大値。 std::numeric_limits<long long>::max() と等しいが、LLONG_MAX は #if などのプリプロセッサディレクティブで使用できる。
具体的な値は実装依存であるが、9223372036854775807（263 - 1）以上であることが規格で定められている。また、型は long long である。

例

#include <climits>
#include <iostream>

int main()
{
  std::cout << LLONG_MAX << '\n';
}

出力例

9223372036854775807
