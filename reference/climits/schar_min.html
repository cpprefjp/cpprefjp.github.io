SCHAR_MIN

implementation-defined#define SCHAR_MIN

概要

signed char の最小値。 値は std::numeric_limits<signed char>::min() と等しいが、型が異なり、また SCHAR_MIN は #if などのプリプロセッサディレクティブで使用できる。
具体的な値は実装依存であるが、-127（-(27 - 1)）以下であることが規格で定められている。また、型は int である。

例

#include <climits>
#include <iostream>

int main()
{
  std::cout << SCHAR_MIN << '\n';
}

出力例

-128
