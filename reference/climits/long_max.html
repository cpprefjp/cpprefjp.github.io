macro
<climits>

LONG_MAX

# define LONG_MAX implementation-defined

概要

long 型が表現できる値の最大値。

std::numeric_limits<long>::max() と等しいが、LONG_MAX は #if などのプリプロセッサディレクティブで使用できる。

具体的な値は実装依存であるが、2147483647（231 - 1）以上であることが規格で定められている。このマクロによって定義される値の型は long である。

例

#include <climits>
#include <iostream>

int main()
{
  std::cout << LONG_MAX << '\n';
}

出力例

2147483647
