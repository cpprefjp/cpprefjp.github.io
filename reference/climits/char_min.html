macro
<climits>

CHAR_MIN

# define CHAR_MIN implementation-defined

概要

char 型が表現できる値の最小値。

値は std::numeric_limits<char>::min() と等しいが、型が異なり、また CHAR_MIN は #if などのプリプロセッサディレクティブで使用できる。

具体的な値は実装依存であるが、char が符号付きであれば SCHAR_MIN と同じ、符号無しであれば 0 である。このマクロによって定義される値の型は int である。

例

#include <climits>
#include <iostream>

int main()
{
  std::cout << CHAR_MIN << '\n';
}

出力例

-128
