infinity

// C++03
static T infinity() throw();

// C++11
static constexpr T infinity() noexcept;

概要

浮動小数点数型における、正の無限表現を取得する。

numeric_limits<float>::has_infinityがtrueのとき、numeric_limits<float>::infinity()はINFINITYマクロの値と等しい。

要件

has_infinity!= false &&is_iec559!= falseこれを満たさない場合はT()が返る。

戻り値

正の無限表現

例外

投げない
