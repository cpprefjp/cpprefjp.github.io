digits

// C++03
static const int digits;

// C++11
static constexpr int digits;

概要

基数 radix において表現できる桁数を示す。整数型であれば、符号ビット以外のビット数である。

浮動小数点数の場合、仮数部の桁数である。

備考

is_specialized == falseの場合は0

対応するマクロを次の表に挙げる。

型 対応するマクロ
float FLT_MANT_DIG
double DBL_MANT_DIG
long double LDBL_MANT_DIG
