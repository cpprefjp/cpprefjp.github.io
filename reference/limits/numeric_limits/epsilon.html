epsilon

// C++03
static const T epsilon() throw();

// C++11
static constexpr T epsilon() noexcept;

概要

機械イプシロンを取得する。
これは浮動小数点数の比較のために誤差幅を評価するのに使用できる。

対応するマクロを次の表に挙げる。

型 対応するマクロ
float FLT_EPSILON
double DBL_EPSILON
long double LDBL_EPSILON
