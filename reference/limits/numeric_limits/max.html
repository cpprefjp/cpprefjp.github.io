max

// C++03
static T max() throw();

// C++11
static constexpr T max() noexcept;

概要

型Tの値の最大値を取得する

戻り値

指定された型の有限値のうち最大のもの。
浮動小数点数の場合、無限大やNaNではない。

例外

投げない

備考

is_specialized == falseの場合はT()が返される。
C++03バージョンはconstexprではないため、非定数式となる。
