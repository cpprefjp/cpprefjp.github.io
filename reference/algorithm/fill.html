概要

要件

value は output iterator へ書き込み可能でなければならない

効果

計算量

正確に last - first 回の代入を行う

実装例

template <class ForwardIterator, class T>
void fill(ForwardIterator first, ForwardIterator last, const T& value) {
  while (first != last)
    *first++ = value;
}
