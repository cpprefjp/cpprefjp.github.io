minmax_element (C++11)

pair<ForwardIterator, ForwardIterator>
    minmax_element(ForwardIterator first, ForwardIterator last);

  template<class ForwardIterator, class Compare>
  pair<ForwardIterator, ForwardIterator>
    minmax_element(ForwardIterator first, ForwardIterator last, Compare comp);
}namespace std {
  template<class ForwardIterator>

概要

[first, last)の範囲において、最小要素を指すイテレータと最大要素を指すイテレータの組を取得する。

戻り値

*i < *jもしくはcomp(*i, *j)の比較によって最小と判断された要素を指すイテレータをfirst、

*j < *iもしくはcomp(*j, *i)の比較によって最大と判断された要素を指すイテレータをsecondに持つpairオブジェクトを返す。

first == lastの場合、make_pair(first, first)を返す。

計算量

nを範囲の要素数とする場合、max(3(n / 2) - 1, 0)回の述語適用を行う。

例

  assert(*i.second == 4);
